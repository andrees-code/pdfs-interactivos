# Fonts Setup (Phase 0)

Place local variable font files here:

- `public/fonts/plus-jakarta-sans/PlusJakartaSans-Variable.woff2`
- `public/fonts/inter/Inter-Variable.woff2`
- `public/fonts/space-grotesk/SpaceGrotesk-Variable.woff2`

These files are referenced by `src/assets/fonts.css`.
If a file is missing, browser falls back to system sans fonts.
